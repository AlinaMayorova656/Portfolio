## Описание проекта

**Тема:** Оценка влияния нового алгоритма рекомендаций на вовлеченность пользователей в развлекательном приложении с короткими видео.

**Проблема:** Необходимо оценить, повышает ли новый алгоритм рекомендаций вовлеченность пользователей в развлекательном приложении с бесконечной лентой. Для этого требуется спланировать A/B-тест, проконтролировать корректность его проведения и проанализировать полученные результаты, чтобы принять обоснованное решение о внедрении алгоритма.

**Инструменты решения**  
- Python: pandas (обработка данных), matplotlib (визуализация)  
- Статистические библиотеки: statsmodels (расчёт мощности, z-тест), scipy.stats (t-тест)  
- Методы: A/B-тестирование, проверка гипотез, анализ однородности групп

**Выполненные задачи:**  
1. Анализ данных (регистрации, просмотры страниц, доля активных пользователей).  
2. Формулировка гипотез и расчёт необходимого размера выборки и длительности теста.  
3. Мониторинг теста: проверка распределения пользователей по группам, отсутствие пересечений, однородность по устройствам и регионам.  
4. Оценка результатов: расчёт основной метрики, проверка статистической значимости изменений, сравнение долей успешных сессий.  
5. Формулировка выводов и рекомендаций.

**Вывод:** Новый алгоритм рекомендаций статистически значимо увеличивает вовлеченность пользователей: зафиксирован прирост ключевой метрики на 1,06% (p-value < 0,05). Тест проведён корректно: группы сопоставимы по размеру (различие менее 1%), пересечения пользователей отсутствуют, распределение по устройствам и регионам однородно.
