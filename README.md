# Portfolio
Моё портфолио учебных проектов по анализу данных.

Здесь собраны реализованные проекты с использованием SQL, Python и статистического анализа.

| # | Наименование проекта | Описание | Стек |
|---|----------------------|----------|------|
| 1 | [Анализ активности пользователей Яндекс Афиши](https://github.com/AlinaMayorova656/Portfolio/tree/main/Project1) | Исследование изменений продаж билетов и пользовательской активности сервиса. Выявление сезонных сдвигов, анализ популярности мероприятий, проверка гипотез о различиях в поведении мобильных и десктопных пользователей. | Jupyter Notebook, Python (pandas, numpy, matplotlib, seaborn), scipy.stats, statsmodels |
| 2 | [A/B-тестирование алгоритма рекомендаций](https://github.com/AlinaMayorova656/Portfolio/tree/main/Project2) | Оценка влияния нового алгоритма на вовлеченность пользователей в приложении с короткими видео. Расчет размера выборки, мониторинг корректности теста, проверка статистической значимости результатов. | Jupyter Notebook, Python (pandas, matplotlib), scipy.stats, statsmodels |
| 3 | [Анализ рынка недвижимости СПб и Ленобласти](https://github.com/AlinaMayorova656/Portfolio/tree/main/Project3) | Исследование объявлений о продаже жилья для агентства из Петрозаводска. Определение ликвидных сегментов, сезонных закономерностей, инвестиционно привлекательных локаций. | PostgreSQL, DBeaver, Jupyter Notebook |
